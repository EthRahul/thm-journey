# 🌐 Network Fundamentals: CCNA 200-301

*A structured knowledge base for networking concepts, derived from the [NetworkChuck Free CCNA Playlist](https://youtube.com/playlist?list=PLIhvC56v63IJVXv0GJcl9vO5Z6znCVb1P).*

## 🎯 Purpose
This directory contains technical notes focused on building a rock-solid foundation in computer networking. While my primary focus is Web Application Penetration Testing and Offensive Security, true exploitation requires a deep understanding of the underlying infrastructure. You cannot effectively compromise, map, or pivot through a network that you do not fundamentally understand.

## ⚔️ The Offensive Perspective
These notes go beyond standard CCNA transcriptions. Each core networking concept is analyzed through a cybersecurity lens to understand:
* **Data Flow & Interception:** How packets travel across routers and switches.
* **Network Segmentation & Pivoting:** How internal subnets are divided and how attackers traverse them.
* **Protocol Vulnerabilities:** The inherent weaknesses in baseline protocols (ARP, DNS, TCP/UDP) that allow for enumeration and exploitation.

## 📂 Documentation Structure
Each file corresponds to a specific networking module and breaks down:
1. **Core Networking Concepts & Hardware**
2. **Real-World Analogies for Complex Protocols**
3. **The Direct Application to Penetration Testing**

*This documentation is continually updated alongside practical lab environments and CTF exercises.*
